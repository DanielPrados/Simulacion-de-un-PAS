# Simulacion de un PAS
Este proyecto contiene la implementación y visualización de un Paseo Aleatorio Simple (PAS). Este es un proceso estocástico en tiempo discreto que toma valores enteros. Concretamente, viene definido como sigue: 

$$X_t=\sum_{s=1}^tZ_s$$

donde las $Z_s$ son variables aleatorias independientes e idénticamente distribuidas de la siguiente forma: 

$$P(X_s=1)=p, \; P(X_s=-1)=1-p$$.
